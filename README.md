# LibraryProject

This is the initial Django project setup for ALX Django Learning Lab.

## Description
LibraryProject is a basic Django project created to demonstrate the setup of a Django development environment, project structure, and running of the Django development server.

## How to Run the Project

1. Install Django:
   pip install django

2. Run migrations:
   python manage.py migrate

3. Start the server:
   python manage.py runserver

4. Open your browser and go to:
   http://127.0.0.1:8000/
